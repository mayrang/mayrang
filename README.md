# 💻 UX와 DX의 공존을 고민하는 프론트엔드 개발자

안녕하세요! **견고한 아키텍처** 위에서 **유려한 사용자 경험**을 그려내고 싶은 개발자입니다.

단순히 기능이 동작하는 것을 넘어, **'왜(Why)'** 이렇게 동작하는지 깊이 파고드는 것을 즐깁니다. SOLID 원칙과 디자인 패턴을 활용해 유지보수하기 좋은 코드를 작성하고, 생동감 있는 인터랙션을 구현하는 데 몰입해 있습니다.

**"사용자에게는 최적의 경험을, 팀원에게는 최고의 효율을"** 이라는 가치를 실현하기 위해 끊임없이 학습하고 소통합니다.
